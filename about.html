Game Instructions üìú

Tic-Tac-Toe is a two player game where each player chooses a set of game pieces.
The first player places their piece in a square and the second one does the same and they each take turn filling in the 3x3 grid.

If a player gets their game pieces in a row(horizontally) they win the game.

If the game pieces are in a column(vertically) they win.

If the game pieces are diagonally placed they also win.
Whether right to left,

or left to right.

If the board is filled with no vertical, horizontal or diagonal wins, the game ends in a tie.
